# rts-ima Query Map

Use the default IMA knowledge base `Rayner Teo 知识库`.

## Query Expansion

- 趋势交易: trend trading, ride trends, trend following, pullback, moving average.
- 价格行为: price action, candlestick, line chart, market structure.
- 支撑阻力: support and resistance, levels, zones, retest.
- 突破: breakout, false breakout, failed breakout, confirmation.
- 风险管理: risk management, stop loss, position sizing, drawdown.
- 交易心理: trading psychology, overtrading, full-time trading, quit trading.
- 工具: TradingView, indicators, moving average, chart pattern.

## Output Discipline

Prefer a small number of high-signal matches. If results are thin, say what is missing and suggest better search terms.
