# rtskill 使用说明

## 这是什么

`rtskill` 是 Rayner Teo 交易教育主题的 Agent Skill 工具箱。它默认连接 `Rayner Teo 知识库`，帮助你学习价格行为、趋势交易、突破、支撑阻力、风控、交易心理，并把资料转成策略研究、交易计划和复盘清单。

## 适合谁

- 想系统学习 Rayner Teo 资料的用户
- 想把交易教育内容整理成笔记、课程或内容的人
- 想把价格行为、趋势、突破、风控转成可执行流程的人
- 想用 IMA 知识库做来源检索和事实核对的人

## 安装方式

```bash
npx -y skills add FocusLiz-Lab/rtskill -g --all
```

## IMA 知识库配置

默认知识库：

```text
Rayner Teo 知识库
```

如果你还没有安装 IMA skill：

```text
请安装 ima 技能
下载地址：https://app-dl.ima.qq.com/skills/ima-skills-1.1.7.zip
API Key 获取：https://ima.qq.com/agent-interface
```

## 常用命令

```text
$rts 用户想把这个知识库用于实际项目，帮他判断应该从哪个模块开始。
```

```text
$rts-learning-map 系统学习这个知识库，先看哪些？
```

```text
$rts-strategy 围绕趋势交易、突破和支撑阻力，整理一套策略假设。
```

```text
$rts-trading-plan 把价格行为方法转成交易前检查清单和复盘表。
```

```text
$rts-review 帮用户点评一份交易计划或交易日志，并给修改建议。
```

```text
$rts-psychology 诊断过度交易、扛单、提前离场和连续亏损后的执行问题。
```

```text
$rts-content 围绕知识库里的核心主题，生成 10 个内容选题。
```

```text
$rts-ima 搜索 Rayner Teo 对 false breakout 的相关讲解。
```

## 典型场景

学习路径：用 `$rts-learning-map` 确定主题顺序和每阶段产出。

策略整理：用 `$rts-strategy` 从 IMA 资料中提炼策略结构，再用 `$rts-trading-plan` 做执行清单。

交易复盘：用 `$rts-review` 检查交易计划、入场、止损、目标和执行偏差，再用 `$rts-psychology` 找纪律问题。

内容创作：用 `$rts-strategy` 整理方法论，再用 `$rts-content` 生成选题、脚本或文章框架。

出处核对：用 `$rts-ima` 搜索 IMA，再把结果交给对应 workflow。

## 常见问题

如果回答没有引用具体资料，通常是 IMA 未安装、未配置或当前账号没有知识库权限。先检查 IMA skill 和 API Key。

如果问题涉及具体买卖点、仓位、收益预期或实时市场判断，skill 只能输出教育性流程和风险检查，不给投资建议或收益承诺。

## 发布边界

本项目不包含 Rayner Teo 原始转录稿、X/Twitter 原文、PDF 书籍或私有资料。具体引用和原文核对必须回到 IMA 知识库。
